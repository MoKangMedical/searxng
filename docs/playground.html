API Usage

This playground calls the SearXNG JSON API. You can use it programmatically:

// JavaScript
const res = await fetch("http://localhost:8888/search?q=medical+AI&format=json&engines=google,bing");
const data = await res.json();
console.log(data.results);

// Python
import requests
r = requests.get("http://localhost:8888/search", params={
    "q": "medical AI",
    "format": "json",
    "engines": "google,bing,arxiv"
})
print(r.json()["results"])

Note: Requires search.formats: [html, json] in settings.yml. See API Docs for full reference.
