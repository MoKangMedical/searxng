GET /search Search across multiple engines

Execute a search query across one or more search engines. Returns aggregated results.

Parameters

NameTypeRequiredDescription
qstringrequiredSearch query string
formatstringoptionalResponse format: html, json, csv, rss (default: html)
enginesstringoptionalComma-separated engine names (e.g., google,bing,arxiv)
categoriesstringoptionalFilter by category: general, images, videos, news, science, it
languagestringoptionalLanguage code (e.g., en, zh-CN, auto)
pagenointegeroptionalPage number for pagination (default: 1)
time_rangestringoptionalFilter by time: day, week, month, year
safesearchintegeroptionalSafe search level: 0, 1, 2

POST /search Search via POST (for long queries)

Same as GET /search but accepts parameters as form data. Useful for very long queries that exceed URL length limits.

Request Body

NameTypeRequiredDescription
qstringrequiredSearch query
formatstringoptionalResponse format
enginesstringoptionalEngine selection
Content-Typeapplication/x-www-form-urlencoded

Error Responses

StatusDescriptionSolution

429Rate limit exceededWait and retry; check limiter config
500Server errorCheck docker logs searxng
502Upstream engine errorEngine may be down; try different engines
