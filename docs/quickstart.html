快速开始

5分钟快速部署SearXNG Medical AI

克隆项目

git clone https://github.com/SearXNG Medical AI - 医疗AI智能搜索引擎/searxng.git
cd searxng

启动服务

./manage.sh start

检查状态

./scripts/status.sh

访问搜索

打开浏览器访问: http://localhost:8888

Python集成

from api.searxng_api import SearXNGClient client = SearXNGClient("http://localhost:8888") # 百度搜索 results = client.search_baidu("医疗AI") # 学术搜索 results = client.search_academic("artificial intelligence")

常用命令

# 启动服务 ./manage.sh start # 停止服务 ./manage.sh stop # 重启服务 ./manage.sh restart # 查看状态 ./manage.sh status # 查看日志 ./manage.sh logs # 测试服务 ./manage.sh test
